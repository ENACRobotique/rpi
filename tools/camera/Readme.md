# Camera

- `calibrateCamera.py` : calibre les [paramètres intrinsèques](https://fr.wikipedia.org/wiki/Calibration_de_cam%C3%A9ra) de la caméra.
- `camera_pos.py` : calibre la position et l'orientation de la caméra par rapport au checkboard de calibration.
- `cam2ecal.py` : envoie les images d'une caméra sur un topic eCAL.
- `ecal_cam_visu.py` : Affiche les images depuis un topic eCAL.
