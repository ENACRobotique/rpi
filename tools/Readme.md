# Tools

Divers outils, n'ayant pas la vocation à être exécutés pendant le match, mais servant à la mise au point du robot.

- Voir [camera/Readme.md](camera/Readme.md) pour les outils liés aux caméras
- `lidar/radarqt.py` : Affiche les données du lidar
- `vl53l5cx/vl53visu.py` : Affiche la matrice 8x8 du capteur de distance VL53L5Cx.
- `set_pid_coef.py` : à utiliser depuis un interpréteur python pour regler les coefs du PID de la base roulante.
- `logs/meas_extractor.py` : Permet d'exporter en CSV ou JSON les données d'un topic enregistrées dans un recording eCAL.
